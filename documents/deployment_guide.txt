개발자 온보딩 시스템 배포 가이드

1. 시스템 요구사항
   - Python 3.8 이상
   - 최소 4GB RAM
   - 10GB 디스크 공간

2. 환경 설정
   - 가상환경 생성: python -m venv venv
   - 가상환경 활성화: source venv/bin/activate (Linux/Mac) 또는 venv\Scripts\activate (Windows)
   - 의존성 설치: pip install -r requirements.txt

3. 환경 변수 설정
   - .env 파일 생성
   - OpenAI API 키 설정
   - Google API 키 설정 (선택사항)

4. 데이터베이스 초기화
   - ChromaDB는 자동으로 초기화됩니다
   - documents 폴더에 문서 파일들을 추가하세요

5. 서버 실행
   - 개발 모드: uvicorn app.main:app --reload
   - 프로덕션 모드: uvicorn app.main:app --host 0.0.0.0 --port 8000

6. 문서 업로드
   - 웹 인터페이스에서 "문서 업로드" 버튼 클릭
   - 또는 API 엔드포인트 직접 호출

7. 모니터링
   - /health 엔드포인트로 시스템 상태 확인
   - 로그 파일 확인
   - API 사용량 모니터링

8. 백업
   - chroma_db 폴더 정기 백업
   - documents 폴더 백업
   - 환경 설정 파일 백업

9. 보안 고려사항
   - API 키 보안 관리
   - 방화벽 설정
   - HTTPS 사용 권장
   - 정기적인 보안 업데이트

10. 성능 최적화
    - 임베딩 모델 캐싱
    - 벡터 데이터베이스 인덱싱
    - CDN 사용 고려
    - 로드 밸런싱 설정
